joe 一个可以模拟 WordStar, Pico和 Emacs的文本编辑器。

joe的安装

运行下面命令安装joe

./configure --sysconfdir=/etc --prefix=/usr &&
make &&
make install

配置文件

Joe使用了几个配置文件。在Joe的man文档里面可以找到他们的信息。这些文件是： /etc/jmacsrc, /etc/joerc, /etc/jpicorc, /etc/jstarrc, 和/etc/rjoerc. Joe同时也会读取 ~/.joerc, 这个文件可以从 /etc/joerc 拷贝过来然后依照自己的喜好作一些改动。

包含的程序

包含 jmacs, joe, jpico, jstar, rjoe 和 termidx.

具体说明

jmacs

jmacs 用来用Emacs模拟模式启动joe的符号链接。

joe

joe 一个可以模拟 WordStar, Pico和 Emacs的文本编辑器。

jpico

jpico 用来用Pico模拟模式启动joe的符号链接。

jstar

jstar 用来用WordStar模拟模式启动joe的符号链接。

rjoe

rjoe 启动joe并只编辑命令行指定的文件的符号链接。

termidx

termidx joe用来生成termcap索引文件的程序。

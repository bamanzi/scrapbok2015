So, the easiest way is actually a one-line-fix:

At the end of the script is this line, which attaches you to the newly created session:
tmux attach-session -t "${TMUX_SESSION_NAME}"
https://github.com/dennishafemann/tmux-cssh/blob/21750733c5b6fa2fe23b9e50ce69d8564f2f742a/tmux-cssh#L314

Just replace it with this, if you always have an already running session, and want to "merge" the new window into your currently active session:
tmux move-window -s "${TMUX_SESSION_NAME}"

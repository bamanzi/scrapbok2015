PuTTYTabManager


Descripción

Herramienta de Windows para administrar varias instancias de PuTTY en una sola ventana mediante pestañas.
PuTTY Tab Manager permite ejecutar múltiples sesiones de PuTTY, conocido cliente de ssh y telnet para windows, en una misma ventana mediante pestañas.

Características

  • Todas las características de PuTTY
  • Interfaz de usuario clara con pestañas
  • Pestañas ilimitadas.
  • Solo es un ejecutable no necesita instalacion.
  • Plataformas Win32, no es necesario MS .NET framework. 
  • Menú de PuTTY para cada sesión abierta.
  • Soporte para KiTTY (kitty.exe) , "fork" (bifurcación) de PuTTY.
  • Soporte para MinTTY (mintty.exe) . terminal emulator
  • Envio de scripts
  • Modo captura automático. Captura todas las sesiones de PuTTY abiertas o por abrir
  • Opción de lanzar sesiones desde linea de comandos (-a <PuTTY command line>)
  • Opción de lanzar sesiones desde un fichero (-f <fichero>)
  • Login automático (Protocolo SSH)
  • Modo Potable  
  • Soporte para transferencias de archivos (WinSCP)
  • Soporte servidor X-Server (Xming , VcXsrv , Portable X-Server, ... ).

Requerimientos

  •  Windows XP/7/10. 
  • - PuTTY 0.60

Instación

 La instalación es muy sencilla, descarga el archivo ZIP desde aqui , descomprimelo en la carpeta que desees y ya puedes disfrutar del programa.

 Login automático 

Una forma sencilla de automatizar un inicio de sesión remoto es proporcionar la contraseña en la pestaña 'Connet to' del dialogo 'Add new tab' . Esto no es recomendable por razones de seguridad .Tenga en cuenta que esta a opción sólo esta disponible cuando se utiliza el protocolo SSH. 


Modo Portable

De forma predeterminada, PuttyTabManager utiliza la base de datos del Registro de Windows para guardar su configuración. Es posible guardarla en un archivo para que no haya que escribir nada en el Registro.

Para ello sólo tienes que  copiar el binario de PuTTY (putty.exe) y crear un archivo llamado puttytm.ini en el mismo directorio donde pusiste el binario de PuttyTabManager ( puttytm.exe) .

Linea de comandos

uso: 

    puttytm.exe [-a <PuTTY command line >]  [-c] [-f <fichero>] 

-a <PuTTY command line>: ejecuta PuTTY con los parámetros indicados en <PuTTY command line>

-c : Borra la información de PuTTYTabManager del registro de Windows.

-f <fichero>: ejecuta PuTTY/KiTTY con las opciones que de le indiquen en el fichero de sesiones (una sesión por casa linea)

ejemplo fichero de sesiones PuTTY: 

 

# ejemplo de fichero de sesiones 

# para mas información : http://the.earth.li/~sgtatham/putty/0.62/htmldoc/Chapter3.html

-load session_name

-ssh host_name

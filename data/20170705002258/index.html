    ** multi-tabbed putty (mtputty)

    主页: http://www.ttyplus.com/multi-tabbed-putty

    最新版本: 1.6 beta (1.6.0.176), 2012/05

    + 标签页右键可显示原putty的窗口菜单
+ 不需要dotnet
+ 支持分屏
+ 支持自动登录
+ 支持备份session （导出为xml文件，自己的格式）
+ 支持kitty

    缺点

    • 不支持含有空格、#等字符的session名（比如openstack#10.71.223.249），这种情况下启动putty失败
    • 会导致putty/kitty不能输入中文

    putty tab manager (puttytm)

    https://sites.google.com/site/macdsite/ (被墙)

    最新版本 0.39 (2015/05)

    • 非常小巧，单一文件，也不需要dotnet
    • 支持备份session （导出为reg文件）
    • 可以attach其它程序窗口进来
    • 可以detach

    缺点:

    • 不支持含有空格、#等字符的session名（比如 openstack#10.71.223.249 ），这种情况下启动putty失败
    • 不支持session名搜索，在session多时不太方便

    putty connection manager (puttycm)

    主页: http://puttycm.free.fr/ (down)

    最新版本: 0.7.1 BETA (build 136), 2009/08

    + 支持备份session （导出为xml文件，自己的格式）
+ 支持显示putty的system menu

    缺点:

    - 需要dotnet
- 仅自己的session才可配置自动登录，从putty导入的不行
- 主页失效，不再能获取到（除非一些软件下载网站）
- bug: 完全无法attach putty窗口 (测试时间 <2013-12-11 周三> 0.7.1.136 + putty-0.61/kitty-0.63)
- 不支持kitty，窗口无法嵌入 （putty 0.61似乎也不行? putty 0.60没有问题)

    superputty

    http://code.google.com/p/superputty/

    1.4.0.4 (2013/05)

    + 开源 
+ Supports local shell via MinTTY or puttycyg
+ 支持分屏
+ 支持备份session （导出为xml文件，自己的格式）
+ 支持导入puttycm的XML格式备份文件

    缺点:

    - 无法显示原putty的窗口菜单，对于kitty而言会缺失很多功能
- 需要dotnet
- 不支持自动登录
- 重新从putty导入session时会出现重复，需要删除后再导入
- 启动有点慢
- 不支持script，但支持同时向多个终端发送同一条命令

    关于SuperPuTTY的补充

    • 无法显示原putty的窗口菜单，对于kitty而言会缺失很多功能 由于可以按住Ctrl键点右键出现程序菜单，所以这个问题不再重要

    • 不支持script，但支持同时向多个终端发送同一条命令

    在1.4.0.7 (26 Sep 2015) 增加了scripting能力，除了支持纯文本，还支持一种它称为SPSL的脚本（里面支持sendline, sleep, sendkey等命令） https://github.com/jimradford/superputty/wiki/SPSL-Reference-%28SuperPuTTY-Scripting-Language%29

    • 不支持自动登录

    好像在1.4.0.8解决了: [[https://github.com/jimradford/superputty/issues/558][Introducing SPSL the SuperPuTTY Scripting Language · Issue #558 · jimradford/superputty]]

  9. Ba Manzi 报告者

    又出来一个 WinSSHTerm http://winsshterm.blogspot.com/ （墙内看下载地址: http://www.freewarefiles.com/WinSSHTerm_program_107606.html )

    集成:

    • 支持putty和kitty
    • 支持调用winscp
    • 支持启动时自动拉起X server （看提示是VcXsrv版本）

    session管理:

    • 不支持导入putty/kitty session，全部connection自己管理，支持目录
    • connection不支持对应putty/kitty session，即没有profile机制
    • 自己对putty提供全局性的设置: Font size, Mouse Paste, Terminal Type, Cursor Type, Cursor Blink, Color Scheme
    • 提供color scheme选项，缺省是自己的WinSSHTerm配色（看着像是solarized dark），另外就是putty缺省配色了。选项是全局性的，不能各个session单独设置

    UI

    • 支持多tab
    • 支持分屏，但固定为品字形（它分别称为window 1, 2, 3），不支持拖放（即一个window内的tab不能拖到另外一个window去）

    便携程序，不需要安装。需要dotnet 3.5。

You can start with the original image burnt to uSD, uninstall XBMC when it is plugged into the pcDuino, and then create a new image from the uSD with the software you used in the initial burn.

Also, have you tried simpleCV?  It is very small.

I have it installed on the XBMC version image, along with VNC and a lot of LAMP server software.  It all fits OK.

I don't have a web-cam but I do have a kinect, and it is displaying a depth map at 2fps.

If you do get it up and running it would be interesting to compare the two.
